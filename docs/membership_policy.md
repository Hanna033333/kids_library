# 멤버십 운영 정책 (최종)

본 문서는 '책자리' 서비스의 회원 시스템 운영 정책 및 UI/UX 가이드라인을 정의합니다.

## 1. 핵심 철학
- **UI/UX Consistency**: 사용자가 제공한 3단계 가입 흐름(뤼튼 스타일)을 충실히 구현하여 직관적인 경험을 제공합니다.
- **Hurdle-Free**: 가입 시 본인인증을 제외하여 전환율을 높입니다.
- **Premium Aesthetic**: 가독성이 높은 화이트 테마와 현대적인 디자인 요소를 적용합니다.

## 2. 가입 및 로그인 플로우

### [Step 1] 가입 수단 선택
- **UI**: 중앙에 '책자리' 로고 배치. 
- **버튼**: 
  - 카카오 계정으로 회원가입 (옐로우, 로고 포함, '빠른 회원가입' 배지)
  - 구글 계정으로 회원가입 (화이트, 로고 포함, 테두리 권장)
- **하단 링크**: 아이디 찾기 | 비밀번호 찾기

### [Step 2] 약관 동의
- **내용**: 뤼튼 스타일의 체크박스 리스트 적용.
- **체크 항목**:
  - [ ] 모두 동의
  - [ ] 만 14세 이상입니다 (필수)
  - [ ] [필수] 서비스 이용약관
  - [ ] [필수] 개인정보 수집 및 이용
  - [ ] [선택] 이벤트/혜택 정보 수신 및 활용 동의
- **버튼**: '다음' (동의 완료 시 활성화)

### [Step 3] 비밀번호 설정
- **내용**: 안전한 서비스 이용을 위한 비밀번호 입력 단계.
- **UI**: 입력 필드 하단에 가이드 문구 노출 ("8자 이상, 문자·숫자·특수문자 중 2가지 포함")
- **버튼**: '완료' 또는 '다음'

## 3. 회원 및 데이터 정책

### A. 회원 구분
- **일반 회원**: 소셜 로그인 또는 이메일 가입을 완료한 상태. (찜하기, 검색 등 모든 기본 기능 사용 가능)

### B. 본인인증(CI) 시점
- 가입 및 서비스 이용 단계에서 본인인증(CI 수집)은 현재 고려하지 않습니다. 향후 필요 시점에 재논의합니다.

### C. 운영 정책
- **변경**: 닉네임, 프로필 등 일반 정보 변경 가능. 가입 이메일은 변경 제한.
- **탈퇴**: 신청 즉시 정보 파기.
- **연령**: 만 14세 이상 가입 가능.

## 4. 디자인 가이드 (Visuals)
- **Primary Color**: `#FF4D00` (Branded Orange)
- **Background**: `#FFFFFF` (White)
- **Typography**: Inter 또는 Pretendard (가독성 중점)
- **Button**: `border-radius: 8px`, 고대비 텍스트 적용.
