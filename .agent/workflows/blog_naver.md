# 📝 /blog-naver 네이버 블로그 원고 생성 및 검증 워크플로우

이 워크플로우는 사용자가 `/blog-naver`를 호출했을 때, 또는 수동으로 네이버 블로그 마케팅용 큐레이션 원고를 발행하고자 할 때, 원고 생성 스크립트를 구동하고 마케팅/디자인 표준 규격에 맞는지 검증 및 발행 단계를 제안하는 마스터 지침입니다.

---

## 🛠️ [Phase 1] 환경변수 및 준비사항 검증 (Pre-requisites)

블로그 원고 자동 생성 스크립트는 **Gemini API** 및 **Supabase DB** 연동이 필수적입니다.

1. **환경변수 파일 점검 (`backend/.env`)**
   - 아래의 필수 키들이 올바르게 등록되어 있는지 검사합니다.
     * `GEMINI_API_KEY`: 원고 생성 및 문체 다듬기에 사용되는 API 키
     * `SUPABASE_URL` / `SUPABASE_KEY`: 도서 및 큐레이션 데이터베이스 조회를 위한 Supabase 키
   - *주의*: 해당 파일이 Git Staging 영역에 포함되어 있지 않은지(유출 위험 방지) 재차 확인합니다.

---

## ⚙️ [Phase 2] 원고 생성 스크립트 실행 (Generation)

백엔드 폴더 내에 작성된 `generate_blog_post.py` 스크립트를 실행하여 원고를 자동 생성합니다.

1. **KST 기준 오늘 요일에 매칭되는 큐레이션 원고 자동 생성**
   ```bash
   # 가상환경 활성화 상태에서 실행
   python backend/generate_blog_post.py
   ```
   * **요일별 자동 매핑 규칙**: 
     - 월요일: 1번째 큐레이션
     - 수요일: 2번째 큐레이션
     - 금요일: 3번째 큐레이션
     - 그 외 요일: 1번째(월요일) 큐레이션 폴백

2. **특정 큐레이션 태그를 지정하여 생성**
   ```bash
   python backend/generate_blog_post.py --tag "사회성"
   ```

3. **결과 확인**
   - 생성된 원고는 `/Users/1004823/Desktop/kids_library/backend/` 디렉토리 또는 세션 아티팩트 폴더에 `naver_blog_post_{tag}.md` 형태로 저장됩니다.

## 🎨 [Phase 3] 마케팅 및 UX 규격 준수성 검사 (Marketing & UX Checklist)

작성된 원고가 네이버 블로그 마케팅 표준 가이드라인에 부합하는지 꼼꼼하게 교차 검수합니다.

- [ ] **모바일 가독성 줄바꿈 스타일 (★최중요★)**
  - 스마트폰 화면에서의 스크롤 편의성과 텍스트 가독성을 극대화하기 위해, **1~2개의 짧은 문장(혹은 구/절 단위)마다 반드시 줄바꿈(엔터)과 빈 줄(개행)**을 넉넉히 배치했는지 확인합니다.
  - 모바일 가로폭이 좁은 화면에서 의미가 잘리지 않고 답답하지 않도록 구성해야 합니다.
- [ ] **도서 이미지 마크다운 노출**
  - 본문에 책 표지 이미지가 마크다운 양식(`![도서명 표지](image_url)`)으로 빠짐없이 삽입되어 스마트에디터 ONE 붙여넣기 시 올바르게 노출되는지 확인합니다.
- [ ] **도서 상세 링크 UTM 파라미터 적용**
  - 개별 도서 상세 페이지로 넘어가는 모든 링크에 네이버 블로그 전용 UTM이 누락 없이 설계되었는지 확인합니다:
    * 예시: `https://checkjari.com/book/{book_id}?utm_source=naver_blog&utm_medium=referral&utm_campaign=weekly_{tag_slug}`
- [ ] **큐레이션 전체 목록 링크 및 앵커 텍스트**
  - 원고의 하단에 아래 정형화된 형태의 앵커 텍스트와 랜딩 링크가 영어 슬러그 기반으로 포함되어 있는지 확인합니다:
    * `👉 [더 많은 독서 목록을 보고 싶으시면 여기를 클릭하세요](https://checkjari.com/collections/curation/{tag_slug}?utm_source=naver_blog&utm_medium=referral&utm_campaign=weekly_{tag_slug})`
- [ ] **저자 정보 생략 및 출판사 정보만 단독 노출**
  - 텍스트가 지저분해지는 것을 막고 가독성을 올리기 위해 저자/역자 정보는 제외하고 `📖 도서명 (출판사)` 규격만 적용했는지 확인합니다.
- [ ] **브랜드 아웃트로 문구 및 북마크 유도**
  - 로그인 없이 3초 만에 내 도서관을 설정하여 청구기호와 대출 가능 여부를 확인하는 '책자리'의 와우 포인트를 명시했는지 확인합니다.
  - 포스팅 저장을 권장하고 주변 양육자들과의 카카오톡 공유를 유도하는 소셜 프루프 멘트가 포함되어 있는지 확인합니다.

## 🚀 [Phase 4] 네이버 블로그 스마트에디터 ONE 발행 (Publishing)

1. 생성된 `naver_blog_post_{tag}.md` 파일의 전체 텍스트를 복사합니다.
2. 네이버 블로그 글쓰기 페이지(스마트에디터 ONE)에 붙여넣기합니다.
3. 표지 이미지와 링크 하이퍼링크가 정상적으로 연동되어 나타나는지 최종 비주얼을 점검하고 **[발행]**합니다.
